# 1.静态库和动态库

## 1.静态库

1. 准备程序

add.cpp
```cpp
#include "add.h"

int add(int a,int b){return a+b;}
```

add.h
```cpp
int add(int a,int b);
```

main.cpp
```cpp
#include<iostream>

#include "add.h"

using namespace std;

  

int main(){

    cout<<add(3,5)<<endl;

    return 0;

}
```
