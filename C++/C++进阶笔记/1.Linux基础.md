# 1.静态库和动态库

## 1.静态库

1. 准备程序

add.cpp
```cpp
#include "add.h"

int add(int a,int b){return a+b;}
```

add.h
```cpp
int add(int a,int b);
```

main.cpp
```cpp
#include<iostream>

#include "add.h"

using namespace std;

  

int main(){

    cout<<add(3,5)<<endl;

    return 0;

}
```

2. 生成二进制文件

```
//生成add.o二进制文件，而不是生成可执行文件
g++ -c add.cpp -o add.o
```

3. 创建静态库

在linux中静态库后缀名为`***.a`

```
//生成.a静态库，如果链接多个.o文件，直接在后面添加
ar crs libadd.a add.o
```

4. 使用静态库

```
//main.cpp：想要编译的程序文件
//-L 文件路径：找到静态库的路径，.表示当前路径
//-l库名称：使用库，库名称需要去掉前缀lib和后缀.a
//-I 可执行文件名称：生成可执行文件，文件名称自定
g++ main.cpp -L. -ladd -I main
```


## 2.动态库

1. 准备程序文件（如上）

2. 生成二进制文件
```
g++ -fPIC -c add.cpp -o add.o
```

3. 创建动态库

```
g++ -shared add.o -o libadd.so
```

也可以2 3步一起做
```
g++ -fPIC -shared add.cpp -o libadd.so
```

4. 使用动态库
```
//和静态库的使用相同
g++ main.cpp -L. -ladd -I main
```
