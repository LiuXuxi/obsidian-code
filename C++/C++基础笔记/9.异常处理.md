- 自定义异常
	- 通过继承和重载exception类来定义新的异常
	- ```
	  #include<iostream>
	  #include<exception>
	  
	  class MyException:public exception{
		public:
			const char* what()const throw(){
				return "my test exception";
			}
		  
	  }
	  //what()是一个由exception类提供的公共方法。它用于返回异常的原因
	  ```
