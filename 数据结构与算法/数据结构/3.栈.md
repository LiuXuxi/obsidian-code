# 维护右枚举左

[3412. 计算字符串的镜像分数 - 力扣（LeetCode）](https://leetcode.cn/problems/find-mirror-score-of-a-string/submissions/720713697/)
>维护所有含有相同Key值的Val，但查找一个符合要求的val：
> 先进先出：stack
> 先进后出：queue
> 需要维护多个结构，可以用哈希表也可以用数组

哈希表维护：
```cpp
class Solution {

public:

    char mirror(char&c){

        return (25-(c-'a'))+'a';

    }

  

    long long calculateScore(string s) {

        unordered_map<char,stack<int>>cnt;

        int n=s.size();

        long long ans=0;

        for(int i=0;i<n;i++){

            char val=mirror(s[i]);

            if(cnt.contains(val)){

                ans+=i-cnt[val].top();

                cnt.erase(val);

            }else cnt[s[i]].push(i);

        }

        return ans;

    }

};
```
数组维护：
```cpp
class Solution {
public:
    long long calculateScore(string s) {
        stack<int> stk[26];
        long long ans = 0;
        for (int i = 0; i < s.size(); i++) {
            int c = s[i] - 'a';
            if (!stk[25 - c].empty()) {
                ans += i - stk[25 - c].top();
                stk[25 - c].pop();
            } else {
                stk[c].push(i);                
            }
        }
        return ans;
    }
};
```
