# 1.整数相凑


[1128. 等价多米诺骨牌对的数量 - 力扣（LeetCode）](https://leetcode.cn/problems/number-of-equivalent-domino-pairs/submissions/705395356/)

> **关键点：** 二元组的每个值不超过10。
> 哈希表的键值不能为数组，该如何表示键值？
> 将数组表示为数值，例如二元组的两个数拼凑起来，这样同样符合（满足相同条件的值所对应的键值相同的）键值的条件。

```cpp
class Solution {

public:

    int numEquivDominoPairs(vector<vector<int>>& dominoes) {

        vector<int>cnt(100,0);

        int ans=0;

        for(auto&dominoy:dominoes){

            int val=dominoy[0]<=dominoy[1]?dominoy[0]*10+dominoy[1]:dominoy[1]*10+dominoy[0];

            ans+=cnt[val];

            cnt[val]++;

        }

        return ans;

    }

};
```


# 2.分数->浮点数


[2001. 可互换矩形的组数 - 力扣（LeetCode）](https://leetcode.cn/problems/number-of-pairs-of-interchangeable-rectangles/)
> 哈希表的键值不能为数组，该如何表示键值？
> 唯一性为`height/weight`，但注意：用`double`存储和计算键值，而不是`int`。
```cpp
class Solution {

public:

    long long interchangeableRectangles(vector<vector<int>>& rectangles) {

        unordered_map<double,int>cnt;

        long long ans=0;

        for(auto&rectangle:rectangles){

            double value=(double)rectangle[1]/rectangle[0];

            ans+=cnt[value];

            cnt[value]++;

        }

        return ans;

    }

};
```

# 3.pair<int,int>->long long


[3755. 最大平衡异或子数组的长度 - 力扣（LeetCode）](https://leetcode.cn/problems/find-maximum-balanced-xor-subarray-length/submissions/708441976/)

> 如何表示二元组pair<eor,sum>？
> 注意sum不能为负。
```cpp
class Solution {

public:

    int maxBalancedSubarray(vector<int>& nums) {

        int n=nums.size();

        unordered_map<long long,int>cnt;

        int eor=0,sum=n;

        int ans=0;

        for(int i=0;i<n;i++){

            eor^=nums[i];

            sum+=nums[i]%2?1:-1;

            long long res=((1LL*eor)<<32)|sum;

            if(cnt.contains(res))ans=max(ans,i-cnt[res]);

            else cnt[res]=i;

        }

        return ans;

    }

};
```


[3728. 边界与内部和相等的稳定子数组 - 力扣（LeetCode）](https://leetcode.cn/problems/stable-subarrays-with-equal-boundary-and-interior-sum/solutions/3815641/qian-zhui-he-yu-ha-xi-biao-shi-zi-bian-x-d6vf/)

> 自定义哈希

```cpp
struct TupleHash {

    template<typename T>

    static void hash_combine(size_t& seed, const T& v) {

        // 参考 boost::hash_combine

        seed ^= hash<T>()(v) + 0x9e3779b9 + (seed << 6) + (seed >> 2);

    }

  

    template<typename Tuple, size_t Index = 0>

    static void hash_tuple(size_t& seed, const Tuple& t) {

        if constexpr (Index < tuple_size_v<Tuple>) {

            hash_combine(seed, get<Index>(t));

            hash_tuple<Tuple, Index + 1>(seed, t);

        }

    }

  

    template<typename... Ts>

    size_t operator()(const tuple<Ts...>& t) const {

        size_t seed = 0;

        hash_tuple(seed, t);

        return seed;

    }

};

  

class Solution {

public:

    long long countStableSubarrays(vector<int>& capacity) {

        unordered_map<tuple<int, long long>, int, TupleHash> cnt;

        long long sum = capacity[0]; // 前缀和

        long long ans = 0;

        for (int r = 1; r < capacity.size(); r++) {

            ans += cnt[{capacity[r], sum}];

            cnt[{capacity[r - 1], capacity[r - 1] + sum}]++;

            sum += capacity[r];

        }

        return ans;

    }

};
```
