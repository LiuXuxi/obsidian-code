# 质因子分解

> **将一个合数（大于1的整数）分解成一系列质数相乘的过程**

```cpp
class Solution {

public:

    int countPrimes(int n) {

        if(n<=1)return 0;

        int ans=0;

        for(int i=2;i*i<=n;i++){

            if(n%i==0){

                ++ans;

                while(n%i==0)n/=i;

            }

        }

        if(n>1)++ans;

        return ans;

    }

};
```
